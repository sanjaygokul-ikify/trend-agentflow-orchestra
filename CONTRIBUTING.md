# Contributing to AgentFlow Orchestra

## Development Setup
1. Clone the repo: `git clone https://github.com/your/repo.git`
2. Set up dev environment: `make dev`
3. Run tests: `make test`

## Contribution Process
1. Create a clear RFC in docs/rfcs/
2. Implement with full unit tests
3. Provide performance metrics (before/after)
4. Update docs with new usage patterns

## Supported Platforms
- x86_64 Linux (tested on Ubuntu 22.04+)
- Apple Silicon (Rosetta 2)
- Kubernetes 1.24+
